| # | 类型 | 文件 | 问题 |
|---|------|------|------|
| 1 | **新发现** | `base-embedding.ts:28-29` | **`preprocessText` 使用 4 chars/token 近似截断**：`const maxChars = this.maxTokens * 4` 对中文/日文/韩文等 CJK 语言偏保守（每字符约 1-2 个 token），对英文偏宽松（每单词约 1-2 个 token，但平均 4 字符）。极端情况下，中文文本可能误截断有效内容，英文文本可能超出 token 限制。 |
| 2 | **新发现** | `extractor.ts:259-270` vs `graph-handlers.ts:835-848` | **`extToLanguage` 遗漏 `.mjs` 扩展名**：`langToExts` 对 javascript 返回 `.js`、`.jsx`、`.mjs`，但 `extToLanguage` 仅映射 `.js`/`.jsx`，未映射 `.mjs`。导致 `.mjs` 文件被扫描到但 `extToLanguage` 返回 `''`，在 `handleIndexRepository` 中被跳过。 |
| 3 | **新发现** | `graph-handlers.ts:467-485` | **`handleDetectChanges` git diff 语义不一致**（前次未修复）：`git diff --name-only ${getDiffBranch}...HEAD` 失败时 fallback 到 `git diff --name-only HEAD`。 |
| 4 | **新发现** | `sync.ts:171` | **`getIndexedCodebases()` 日志打印 identities 而非 localPath**（前次未修复）：`console.log` 打印 `indexedCodebases` 数组内容为 identities。 |
